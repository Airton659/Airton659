## 🚀 Projetos em Destaque

### 🗂️ GATI — Gestão de Ativos de TI · [FUNCESI]
Sistema full stack em produção para gestão de ativos, custódia e rastreabilidade de equipamentos de TI.

- **Frontend:** React 19 + TypeScript + Vite + TailwindCSS + TanStack Query
- **Backend:** C# .NET 8 ASP.NET Core + Dapper + SQL Server
- **Infra:** Docker + Portainer + GitHub Actions + ghcr.io em servidor Linux
- **Destaques:** Módulo de custódia com troca de responsabilidade (senha + foto obrigatória), PDFs automáticos via **QuestPDF**, assinatura eletrônica (Canvas API), RBAC granular e Audit Trail com diff em JSON
- **Testes:** xUnit + Moq + FluentAssertions (backend) · Jest + React Testing Library (frontend) — badge **passing** no CI

---

### 📊 Dashboard Estratégico de TI · [FUNCESI]
Monitoramento em tempo real de KPIs de Service Desk exibido em Smart TVs institucionais.

- Desenvolvido em **Node.js/Express/Socket.io** e migrado para **C# .NET 8 + Microsoft SignalR** (atualizações bidirecionais sem polling)
- KPIs: FCR Same-Day, Auto-Close Tracker (3 dias), produção mensal com cutoff dinâmico e department mapping
- Interface em **Vanilla JS + TailwindCSS** com toggle Daily Ops / YTD
- **Testes:** xUnit + Moq (backend) · Playwright E2E (frontend) — badge **passing** no CI

---

### 🏛️ Plataforma Secretaria — Créditos Curriculares · [FUNCESI]
SPA integrada ao ERP **TOTVS RM** via WebService SOAP e conexão ODBC.

- Virtual DOM customizado em **Vanilla JS** (zero dependências no frontend)
- Queries com até **7 JOINs** em instâncias TOTVS, mapeamento hierárquico tridimensional **O(1)** em C#
- In-Memory Caching extensivo + **SignalR** para estado concorrente entre coordenadores
- Impressão via CSS `@media print` sem bibliotecas externas

---

### 📊 SAD — Sistema de Avaliação de Desempenho · [Itaurb]
PWA completo em Flutter/Dart com backend Firebase, cobrindo o ciclo completo de avaliação de desempenho.

- **Stack:** Flutter · Dart · Provider · Firebase Firestore · Firebase Hosting · Firebase Storage · googleapis
- **Destaques:** Assinatura digital segura com geolocalização, hash SHA-256 e metadados do dispositivo; geração de PDFs (Anexos I a XI) diretamente pelo app; integração com Google Sheets; central de ciclos avaliativos e impressão em lote com filtros
- **Testes:** Playwright E2E — badge **passing** no CI
